# GifFresh
-----------
GifFresh allows you to see trending gifs, It also allows you to search gifs and save to favourite.


## Tools
--------
- Android Studio (version 4.2.2)
- Kotlin (version 1.5.21)
- Gradle (version 4.2.2)
- Github
